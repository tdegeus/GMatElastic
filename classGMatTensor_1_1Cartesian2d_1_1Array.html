Detailed Description

template<size_t N>
class GMatTensor::Cartesian2d::Array< N >

Array of tensors:

  • scalars: shape [...].
  • 2nd-order tensors: shape [..., 2, 2].
  • 4nd-order tensors: shape [..., 2, 2, 2, 2].
Template Parameters
NThe rank of the array (the actual rank is increased with the tensor-rank).
